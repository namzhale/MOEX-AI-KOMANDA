# Data And Model Resources

## Polza.ai

The PDF suggests, but does not require, using `https://polza.ai/` as a model aggregator.

Purpose:

- access to modern LLMs;
- API requests to models through Polza.ai.

Known documentation URL:

```text
https://polza.ai/docs
```

Known team budget from the PDF:

- 6000 RUB for Stage 1;
- another 6000 RUB automatically on transition from Stage 1 to Stage 2.

Known local env variable from the team:

```text
POLZA_API_KEY
```

## MOEX ALGOPACK

The PDF suggests, but does not require, using ALGOPACK as an open stock dataset.

Purpose:

- market data;
- analytics;
- online trading data.

The PDF says access can be requested through MOEX data support.

Keep registration/access steps outside project docs unless needed operationally.
